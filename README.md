# EcoScan 🌱

EcoScan is a lightweight environmental scanning tool designed to help users quickly identify and understand the environmental impact of everyday products and materials.

## 🚀 Features
- Fast scanning and classification of items
- Simple, clean user interface
- Designed for quick environmental insights
- Lightweight and easy to run


## Getting Started
```bash
git clone https://github.com/pranavsan/EcoScan
cd EcoScan
npm install   # or pip install -r requirements.txt
npm start     # or python app.py
